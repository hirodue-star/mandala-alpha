# 音源ファイルの追加方法

以下の音源ファイルをこのフォルダに追加してください。

## 必要ファイル（無料音源）
- track_0.mp3 〜 track_7.mp3 : 各楽器のループ音源（8〜16秒、同一BPM）
- orchestra.mp3               : フルオーケストラ演奏（孵化演出用、15〜30秒）

## 無料音源の入手先（コスト0）
- Freesound.org (CC0ライセンス)
- DOVA-SYNDROME https://dova-s.jp/bgm/
- 魔王魂 https://maoudamashii.jokersounds.com/
